# 🌳 Decision Tree Classifier

A machine learning project that builds and visualizes a Decision Tree Classifier using entropy-based splitting with scikit-learn.

## 📊 Project Overview

This project implements a Decision Tree classification model. The tree is fully visualized showing entropy values, sample counts, and class predictions at each node — making it easy to understand how the model makes decisions.

## 📸 Sample Output

> Full decision tree visualization with color-coded nodes (orange = majority class NO, blue = majority class YES), entropy values, and sample distributions at each split.

## 🛠️ Tech Stack

- **Python**
- **Pandas** — data preprocessing
- **scikit-learn** — Decision Tree model, train/test split
- **Matplotlib** — tree visualization

## ⚙️ How to Run

1. Clone the repository
```bash
git clone https://github.com/Ankush0004/decision-tree-classifier-sklearn.git
cd decision-tree-classifier-sklearn
```

2. Install dependencies
```bash
pip install pandas scikit-learn matplotlib
```

3. Run the notebook
```bash
jupyter notebook task3.ipynb
```

## 📌 Key Features

- Entropy-based splitting (information gain)
- Full tree visualization using `plot_tree()`
- Confusion matrix and accuracy evaluation
- Color-coded nodes for easy class identification
